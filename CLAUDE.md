# product-development (감정 오브제 명상 앱)

Flutter 앱. 정본 기획·디자인 명세: `docs/PRODUCT_SPEC.md`.
경험 루프: **Soothe(오브제 교감) → Pour(글쓰기) → Release(파괴/간직 의식) → Closing**.
> 현재 `lib/`에는 초기 프로토타입(다크 인디고 "흘려보내기") 흔적이 있으나, 목표 제품은 라벤더-핑크 파스텔 claymorphism + 햅틱/센서 중심이다. 신규 작업은 `docs/PRODUCT_SPEC.md`를 기준으로 한다.
